Yatzy Game Project

Interactive Dice Game built with HTML, CSS, and JavaScript

Project Overview

The Yatzy Game was developed as part of Assignments #1 and #2 in my Intro to Internet and Web Development course. It replicates the popular dice game Yatzy, allowing players to roll dice, hold combinations, and calculate scores dynamically. The goal was to combine logic programming, UI feedback, and responsive design into a cohesive experience.

Technical Documentation

  • Frontend Technologies: HTML5, CSS3, JavaScript (ES6)
  • Dynamic UI: JavaScript DOM manipulation for dice rolls and score updates
  • Game Logic: Random number generation with conditional scoring categories
  • Data Handling: Arrays and objects to store game state efficiently
  • Responsive Design: CSS Flexbox and media queries for cross-device compatibility

Technical Highlights

The Yatzy logic utilizes Math.random() to simulate dice rolls, dynamically updates dice visuals, and maintains user scores using object-based state management. Event listeners capture user interactions for rolling, holding, and resetting dice. 


const rollDice = () => Math.floor(Math.random() * 6) + 1;

document.querySelector("#roll").addEventListener("click", () => {
  const results = Array.from({length: 5}, rollDice);
  console.log("Rolled:", results);
});

Technologies & Skills Demonstrated

  • Responsive web design principles (flexbox, grid, and media queries)
  • Separation of concerns (HTML for structure, CSS for styling, JS for interactivity)
  • Integration of Git/GitHub for version control
  • Adherence to HTML & CSS best practices (validated and commented code)
