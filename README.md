## Yatzy Game Overview

The **Yatzy Game** is a browser-based dice game implemented using **HTML, CSS, and JavaScript**.  
It allows players to roll dice, select combinations, and accumulate points — following the traditional Yatzy rules.

### Key Features
- Dice rolling logic implemented with JavaScript randomization  
- Scoring system based on Yatzy rules  
- State management for turns and rounds  
- Responsive design and UI enhancements  
- Clean separation between logic and interface  

### Technologies Used
- **HTML5** – semantic structure and accessible markup  
- **CSS3 / Flexbox / Grid** – responsive and modern layout design  
- **JavaScript (ES6)** – dynamic game logic and DOM interaction  
- **Version Control (Git/GitHub)** – tracked changes and organized commits  
- **Visual Studio Code** – main development environment
